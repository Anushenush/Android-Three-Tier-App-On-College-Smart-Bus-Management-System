🚀 Smart College Bus Management System
A Real-Time Multi-App Transportation Ecosystem (Admin | Student | Driver)
---------------------
⚡ Overview
This project transforms traditional college bus management into a smart, real-time, and connected system.
Instead of manual coordination and WhatsApp updates, this system provides live tracking, transparent fee management, and seamless communication across all stakeholders.
---------------------
🎯 Problem
No real-time bus tracking
Students miss buses due to lack of information
Dependency on WhatsApp for updates
No visibility of alternative buses
Lack of transparency in fee management
No centralized system for control
---------------------
💡 Solution
A Three-Tier Smart System with 3 interconnected Android applications:
---
Admin App → Central control & management
Student App → Real-time access & visibility
Driver App → Live location provider

All apps are connected via Firebase, enabling instant data synchronization.
---------------------
📱 Applications & Features
----
👨‍💼 Admin App
Manage students and drivers
Update fee details with receipts
Monitor all buses in real-time
Centralized system control
---------------------
🎓 Student App
Real-time bus tracking
View fee status & history
Access driver contact details
Find alternative buses if missed
---------------------
🚍 Driver App
One-tap start for tracking
Continuous GPS location updates
Real-time data transmission
---------------------
⚙️ Tech Stack
Frontend: Kotlin (Android)
Architecture: MVVM
Backend: Firebase
Cloud Firestore
Realtime Database
Authentication
Maps & Location: Google Maps SDK, Fused Location Provider
---------------------
